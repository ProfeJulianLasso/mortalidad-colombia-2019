# Capturas para el README

Esta carpeta aloja las capturas de pantalla de la aplicación desplegada. Para que las imágenes se rendericen correctamente en `README.md`, los archivos deben conservar exactamente estos nombres:

| Archivo | Visualización |
|---|---|
| `01-mapa-departamentos.png` | Mapa coroplético de defunciones por departamento (alcance nacional) |
| `02-linea-mensual.png` | Línea de defunciones mensuales (alcance nacional) |
| `03-barras-violencia.png` | Top cinco ciudades por homicidios CIE-10 X95 |
| `04-pie-baja-mortalidad.png` | Diez municipios con menor mortalidad registrada |
| `05-tabla-causas.png` | Tabla de las diez principales causas de muerte |
| `06-barras-apiladas-sexo.png` | Barras apiladas por sexo y departamento |
| `07-histograma-ciclo-vital.png` | Histograma por etapa del ciclo vital |

## Recomendaciones para tomar las capturas

- Capturar desde la URL pública de Render una vez la aplicación esté desplegada, no en local.
- Usar un navegador en ventana de incógnito con tema claro (las visualizaciones están diseñadas sobre fondo blanco).
- Resolución sugerida: 1440 × 900 o superior.
- Para las visualizaciones reactivas (línea, barras X95, pastel, tabla, histograma) tomar las capturas con el filtro "Todos los departamentos" para que los hallazgos coincidan con la interpretación del informe. Si se desean ejemplos adicionales con un departamento específico, agregar como `XX-bonus-<nombre>.png` y referenciarlos al final del README.
